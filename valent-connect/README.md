# Noctalia Valent Connect

A Plugin integrating your mobile devices into a panel using Valent

Requires `valent` to be running which can be installed by setting up the official Valent app.

> [!NOTE]
> For the "Browse Device Files" (SFTP) feature to work seamlessly, ensure you have an active `ssh-agent` and an askpass utility installed (like `ksshaskpass` for Wayland).

On NixOS requires `pkgs.kdePackages.qttools` to be installed explicitly.
